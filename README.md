# CUICurate
A GraphRAG-based Framework for Automated Clinical Concept Curation for NLP applications

CUI-Curate is a python-based tool to support transparent and reproducible clinical NLP through automation of UMLS concept sets curation.

Read the paper: https://arxiv.org/abs/2602.17949
